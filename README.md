# Setup
```bash
# Primeiramente clone esse repositório
$ git clone https://github.com/gustavocozsp/planner.git
```
### 💻 Back-End

```bash
$ cd planner/backend

# Instale as dependências
$ npm install

# Inicie a aplicação
$ npm run dev
```
- Informações API: https://nlw-journey.apidocumentation.com/reference

### 💻 Front-End

```bash
$ cd planner/frontend

# Instale as dependências
$ npm install

# Inicie a aplicação
$ npm run dev
```
